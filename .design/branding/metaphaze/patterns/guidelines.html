colors

light mode — literal inversion

#fafafa --mz-bg · page, surface, pane
#0a0a0a --mz-fg · body, headlines, code
#6a6a6a --mz-fg-muted · metadata, prompts
#e0e0e0 --mz-border · pane borders, code bg
#2e8b57 --mz-signal · large glyphs only on light (AA)
#b8860b --mz-warn · [WARN] badge
#8b6508 --mz-error · [ERR] (never red)

all primary pairs pass WCAG AA. body text hits 16.97:1 on dark and 18.95:1 on light. the light-mode signal (#2e8b57) is large-glyph only — cursor, status dots, anything at or above 24px.

constraints

the brand is the absences as much as the presences. never on the left, always on the right.

never

  • non-monospace fonts
  • border-radius > 0 anywhere
  • drop shadows or colored shadows
  • background images or photography
  • gradients of any kind
  • pure #000000 or pure #ffffff
  • neon terminal green (#00ff00, #33ff00)
  • dev-tool blue (#3b82f6 and cousins)
  • red for errors (amber only)
  • CRT scanline overlay
  • phosphor text glow
  • typewriter reveal animations
  • ALL CAPS headlines
  • italics
  • smart quotes
  • emoji
  • icon libraries
  • mascots, illustrations, 3D renders
  • backdrop-filter, mix-blend-mode, filter
  • agentic, empower, transform, unlock, leverage

always

  • monospace font on every element
  • lowercase headlines, nav, body, labels
  • bracketed navigation [/about]
  • bracketed buttons [ initiate ]
  • bracketed status codes [OK] [ERR]
  • shell prompt prefixes $ > ~
  • em-dash as structural divider ( — )
  • box-drawing for diagrams (─ │ ┌ ┐ └ ┘)
  • real TUI recordings as hero asset
  • the manifesto preserved verbatim
  • signal color on < 1% of pixels
  • NO_COLOR respected — brand still works
  • straight quotes only
  • zero-radius everywhere
  • 1px solid borders for panes
  • flat surfaces — no overlays
  • step-end easing on the cursor blink
  • one typeface, everywhere
  • weights 400 / 600 / 700 only
  • the readme is the marketing
