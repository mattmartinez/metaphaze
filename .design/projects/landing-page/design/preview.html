§2 what it does

what it does

metaphaze is a command-line orchestrator that drives claude code autonomously through multi-phase software projects. you define the project in a brief. metaphaze runs the phases — research, design, build, review — returning only when the work is done.

the orchestrator runs outside the loop so you can step away from the keyboard and come back to working code.
